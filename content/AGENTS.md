# Content maintenance instructions

Before adding or editing content items, consult the format guide for the affected section and preserve the current Markdown structure exactly.

Section guides:
- Services: `services.format.md`
- Projects: `projects.format.md`
- Team: `team.format.md`
- Research: `research.format.md`
- Multimedia: `multimedia.format.md`
- Press: `press.format.md`

Multimedia rule (sine qua non): every video or podcast entry must include a `- Duration: N min` metadata line calculated from the linked media URL before it can be added or published.
