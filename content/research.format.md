# Research content format

Use `research.md` for published papers. The section starts with the current heading and intro, followed by repeated paper entries.

```md
[**“<Paper title>”**](<url>)
*<i class="fa-solid fa-book-medical"></i> <Journal> • <DD/MM/YYYY> • <Authors>*
```

Rules:
- Keep the title link and metadata line adjacent.
- Use `DD/MM/YYYY` dates for reliable sorting and display.
- Preserve the Font Awesome icon in the metadata line.
