# Team content format

Use `team.md` for team member cards. Each member is separated by `---`.

```md
---

### <emoji> <Full name>
**<Role>**
**IMAGE:** <absolute image URL>
<Short biography paragraph>
*[LinkedIn](<url>) • [ORCID](<url>) • [GitHub](<url>)*
```

Rules:
- Role and image lines are required.
- Use an absolute raw GitHub URL for repository-hosted images instead of a relative `images/...` path.
- Keep the biography as plain text directly after the image line.
- Keep social links in one italic line; omit unavailable profiles rather than leaving empty links.
